# legacy/ — замороженный бекап ванильной версии LUMEN

Это снимок исходников **до** технологической миграции на TypeScript → pixi.js → Electron.

- Состояние соответствует git-тегу **`v1.0-vanilla`**.
- Чистый ванильный JS (ES5-стиль, глобальные объекты), Canvas 2D, без зависимостей.
- Запуск оригинала: открыть `legacy/index.html` в браузере (или отдать любым HTTP-сервером).
- Регресс-харнес оригинала: `legacy/harness.mjs` (читает `legacy/js/*`, см. путь внутри если запускать отдельно).

**Не редактировать.** Папка существует только для отката и сверки поведения «без регрессий».
Актуальная разработка идёт в `src/` (TypeScript). История переноса — в git-логе и тегах фаз.
