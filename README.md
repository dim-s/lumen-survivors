# LUMEN — Survivors

**Сдержи тьму. Стань рассветом.**

Неоновый survivors-like на чистом HTML5 Canvas. Ты — одинокий огонёк света против
наступающей ночи. Без движка, без сборки, без внешних зависимостей и сетевых
загрузок: вся графика процедурная (свет/тьма через glow-спрайты и оверлеи),
звук синтезируется в WebAudio.

## Запуск

```bash
# вариант 1: локальный сервер (рекомендуется)
python3 -m http.server 8177
# открыть http://localhost:8177/index.html

# вариант 2: двойной клик по index.html — работает и на file://
# (классические <script>, без ES-модулей и fetch)
```

## Как играть

- **WASD / стрелки** — движение. Оружие стреляет автоматически.
- В меню: **Пробел** — выбрать героя и начать, **M** — магазин света.
- Убивай тварей ночи → собирай искры → набирай уровень.
- На каждом уровне — **драфт 1 из 3** (`1`/`2`/`3`, стрелки + `Enter`, или клик).
- **Эволюции:** прокачай оружие до 5 ур. + нужную пассивку → в драфте появится
  особая карта **★ ЭВОЛЮЦИЯ ★** (супер-оружие).
- `ESC` / `P` — пауза (сводка билда, громкость `[` `]`, `M` — без звука, `Q` — в меню).
- Выживи 10 минут. Боссы: **Жнец** на 5:00, **Затмение** на 9:30.

## Твист: ТЫ — СВЕТ

Игрок — источник света. Тьма наступает с краёв экрана; твой свет её отталкивает.
**Радиус света растёт с уровнем** (из слабой искры в слепящее солнце) и **проседает
при уроне** (тьма наступает). Враги **вне света быстрее** — рвутся из теней; в свете
замедляются и раскрываются. Это буквальная арка **Хаос → Порядок**.

## Контент

- **6 оружий:** Импульс, Ореол, Нова, Хлыст, Разряд (цепь-молния), Светляк (мины).
- **6 эволюций:** Луч Зари, Корона, Сверхновая, Солнечный Клинок, Гроза, Протуберанец.
- **5 пассивок** + **5 типов врага** (вкл. дальника-Плевуна) + **2 босса**.
- **2 героя:** Искра (равновесие) и Уголёк (стекло-пушка).
- **Мета-прогрессия:** золото за забеги → магазин перм-апгрейдов (HP, урон, скорость,
  свет, второе дыхание). Лучший забег и громкость — в localStorage.

## Дизайн (OGD)

- **Тип:** Игра-Тренажёр (кор-луп забега) + Маятник-фаза драфта.
- **Формула Обещания:** дефицит — бессилие, тебя поглощает тьма; инструмент — стань
  светом, что держит ночь. Считывается за полсекунды (тёплое пламя vs холодная тьма).
- **Транзит:** нулевой порог — двигаешься и стреляешь с первой секунды.
- **Мягкая Посадка:** итоги + золото в магазин → «ещё один забег».

## Тесты

Headless регресс-харнесс — чистый Node, без браузера и зависимостей. Грузит игровые
скрипты в `vm`-контекст с DOM/canvas-заглушкой и гоняет сидированные бот-симуляции,
проверяя инварианты: детерминизм, отсутствие исключений в логике и рендере (все 6
экранов), эскалация, прогрессия, эволюции, дальники-стрелки, оба босса, мета-апгрейды
и второе дыхание, второй персонаж.

```bash
node tests/harness.mjs            # сиды по умолчанию (42, 7, 1234)
node tests/harness.mjs 42 7 99    # свои сиды
# код выхода 0 = PASS, 1 = FAIL
```

Геймплейный рандом детерминирован через `RNG` (mulberry32): в браузере сид задаётся
`?seed=N` (`index.html?seed=42`), иначе по времени.

## Структура кода

```
index.html          — точка входа, подключение скриптов по порядку
css/style.css       — фон, центрирование canvas
js/config.js        — ВСЯ балансировка: статы, кривая эскалации, оружие, эволюции,
                      враги, персонажи, магазин (единственный источник правды)
js/utils.js         — RNG, математика, пул объектов, weaponDef
js/render.js        — кэш неоновых glow-спрайтов
js/audio.js         — синтез SFX + дрон-луп + громкость (WebAudio)
js/input.js         — клавиатура/мышь
js/entities.js      — игрок (статы, свет, мета-апгрейды) + фабрики
js/weapons.js       — поведение оружия (болт/орбита/нова/хлыст/цепь/мины)
js/spawner.js       — спавн по фазам, дальники, боссы
js/game.js          — состояние, апдейт, коллизии, свет/тьма, драфт, эволюции, мета
js/ui.js            — рендер мира + тьмы, HUD, экраны (меню/выбор/магазин/драфт/
                      пауза/итоги/победа)
js/main.js          — DPR-canvas, фикс-таймстеп, маршрутизация ввода, Meta-сохранение
tests/harness.mjs   — headless регресс-харнесс
```

Тюнинг геймплея — только в `js/config.js`.
