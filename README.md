# LUMEN — Survivors

**Сдержи тьму. Стань рассветом.**

▶️ **[Играть онлайн](https://dim-s.github.io/lumen-survivors/)** — браузерная версия (ПК и телефон).
🔒 Исходники открыты для изучения, но это **source-available, не open source** — см. [Лицензию](#лицензия).

Неоновый survivors-like: ты — одинокий огонёк света против наступающей ночи.
Десктопная игра для Steam. Вся графика процедурная (свет/тьма через WebGL-шейдеры
и glow-спрайты), звук синтезируется в WebAudio, прогресс — в localStorage.

**Технологии:** TypeScript · [Vite](https://vitejs.dev) · [pixi.js v8](https://pixijs.com)
(WebGL-рендер мира и тьмы) · [Electron](https://www.electronjs.org) (десктоп-сборка).
Рендер гибридный: pixi/WebGL рисует мир + шейдер тьмы на нижнем канвасе, HUD и меню —
на 2D-оверлее сверху.

## Управление

- **ПК:** WASD / стрелки — движение, мышь/клавиши — меню. Оружие стреляет само.
- **Телефон (веб-сборка):** тащи палец — движение (плавающий джойстик), тап — меню.

## Запуск и сборка

```bash
npm install

# разработка в браузере (быстрый цикл, hot-reload)
npm run dev               # http://localhost:5173  (?seed=N для детерминизма)

# веб-сборка (dist/)
npm run build             # tsc --noEmit + vite build
npm run preview           # отдать собранный dist/

# Electron (десктоп)
npm run electron:dev      # Vite dev-сервер + Electron-окно
npm run electron:run      # собрать и запустить как десктоп-приложение
npm run electron:build    # упаковать под текущую ОС (release/)
npm run electron:build:win    # Windows-установщик (.exe, nsis) — приоритет для Steam
npm run electron:build:mac    # macOS (.dmg)
npm run electron:build:linux  # Linux (AppImage)
```

Артефакты сборки → `release/`. Windows-сборку с другой ОС нужно собирать на Windows
(или через wine), конфиг в поле `build` в `package.json`.

## Как играть

- **WASD / стрелки** — движение. Оружие стреляет автоматически.
- В меню: **Пробел** — выбрать героя и начать, **M** — магазин света.
- Убивай тварей ночи → собирай искры → набирай уровень.
- На каждом уровне — **драфт 1 из 3** (`1`/`2`/`3`, стрелки + `Enter`, или клик).
- **Эволюции:** прокачай оружие до 5 ур. + нужную пассивку → в драфте появится
  особая карта **★ ЭВОЛЮЦИЯ ★** (супер-оружие).
- `ESC` / `P` — пауза (сводка билда, громкость `[` `]`, `M` — без звука, `Q` — в меню).
- Выживи 10 минут. Боссы: **Жнец** на 5:00, **Затмение** на 9:30.

## Твист: ТЫ — СВЕТ

Игрок — источник света. Тьма наступает с краёв экрана; твой свет её отталкивает.
**Радиус света растёт с уровнем** (из слабой искры в слепящее солнце) и **проседает
при уроне** (тьма наступает). Враги **вне света быстрее** — рвутся из теней; в свете
замедляются и раскрываются. Это буквальная арка **Хаос → Порядок**.

## Контент

- **11 оружий:** Импульс, Ореол, Нова, Хлыст, Разряд, Светляк, Луч-маяк (постоянный
  луч-свет), Пульс-фонарь (свет-зона тормозит тьму), Сумеречный клинок (бьёт сильнее
  у кромки тьмы), Отражённый луч (рикошет от границы света), Рассветная вспышка (ульта).
- **11 эволюций** — по одной на оружие (Луч Зари, Корона, Сверхновая, Солнечный Клинок,
  Гроза, Протуберанец, Солнечное Копьё, Маяк, Клинок Затмения, Призма, Рассвет).
- **5 пассивок** + **9 типов твари** (Тень, Морок, Гнёт, Плевун-дальник, Пожиратель —
  высасывает свет, Дробитель — делится, Якорь — роняет пятно тьмы, Рассеиватель —
  глушит свет, Громада) + **4 босса** (Жнец, Затмение, Владыка Затмения, Нулевая Точка).
- **4 героя:** Искра (равновесие), Уголёк (стекло-пушка), Угасающий (свет тает — но
  убийства подливают), Зеркало (свет рождается только из убийств). Двое заперты до вех.

## Реиграбельность (v2)

- **Глубины Тьмы** — ascension-лестница из 6 ступеней: победа открывает следующую,
  каждая стекает модификаторы (враги крепче, свет слабее) и множит награду; на глубине
  меняется ростер боссов. Главный мотор «ещё забег».
- **Аномалии тьмы** — на старте роллятся 1–3 модификатора правил (Сумрак, Долгая ночь,
  Прилив роя, Неистовство, Морок, Угли, Голод, Хрупкость): каждый забег непохож.
- **Маятник День/Ночь** — радиус света дышит, частота растёт к финалу.
- **Развилки Тьмы** — на 3:25 и 6:55 выбор без правильного ответа (Прорыв теней /
  Сгущение тьмы / Затишье / Россыпь).
- **Разблокировки и Кодекс** — оружие и герои открываются за вехи; убитые твари
  заносятся в Кодекс. Прогресс копится между забегами.
- **Мета-прогрессия:** золото за забеги → магазин перм-апгрейдов (HP, урон, скорость,
  свет, второе дыхание). Лучший забег, глубины, разблокировки и громкость — в localStorage.

## Дизайн (OGD)

- **Тип:** Игра-Тренажёр (кор-луп забега) + Маятник-фаза драфта.
- **Формула Обещания:** дефицит — бессилие, тебя поглощает тьма; инструмент — стань
  светом, что держит ночь. Считывается за полсекунды (тёплое пламя vs холодная тьма).
- **Транзит:** нулевой порог — двигаешься и стреляешь с первой секунды.
- **Мягкая Посадка:** итоги + золото в магазин → «ещё один забег».

## Тесты

Headless регресс-харнесс — чистый Node, без браузера. Собирает игровую логику в один
бандл (esbuild) и грузит в `vm`-контекст с DOM/canvas-заглушкой, гоняя сидированные
бот-симуляции и проверяя инварианты: детерминизм, отсутствие исключений в логике и
2D-рендере (HUD/меню), эскалация, прогрессия, эволюции, дальники-стрелки, боссы и их
смерть, мета-апгрейды и второе дыхание, все персонажи. Плюс v2: свет-механики врагов
(высасывание/разделение/пятно тьмы/подавление), детерминизм аномалий, модификаторы
глубин, вехи разблокировок, рикошет, режимы света героев, кольца боссов.

```bash
npm test                          # собрать бандл + прогнать харнесс (сиды 42, 7, 1234)
node tests/build-test-bundle.mjs  # пересобрать только бандл
node tests/harness.mjs 42 7 99    # харнесс со своими сидами
# код выхода 0 = PASS, 1 = FAIL
```

WebGL-рендер мира харнесс не покрывает (нет GPU в Node) — он проверяет логику и 2D-слой;
визуал pixi проверяется вживую. Геймплейный рандом детерминирован через `RNG`
(mulberry32): в браузере сид задаётся `?seed=N`, иначе по времени.

## Структура кода

```
index.html            — точка входа Vite (module-скрипт + два канваса: pixi + 2D-HUD)
css/style.css         — стэк канвасов, оверлей «поверни устройство»
src/config.ts         — ВСЯ балансировка: статы, кривая эскалации, оружие, эволюции,
                        враги, персонажи, магазин (единственный источник правды)
src/utils.ts          — RNG, математика, пул объектов, weaponDef
src/render.ts         — кэш неоновых glow-спрайтов (offscreen-canvas)
src/audio.ts          — синтез SFX + дрон-луп + громкость (WebAudio)
src/input.ts          — клавиатура / мышь / сенсор
src/entities.ts       — игрок (статы, свет, мета-апгрейды) + фабрики пулов
src/weapons.ts        — поведение оружия
src/spawner.ts        — спавн по фазам, дальники, боссы
src/game.ts           — состояние, апдейт, коллизии, свет/тьма, драфт, эволюции
src/meta.ts           — мета-прогрессия (localStorage)
src/ui.ts             — 2D-ОВЕРЛЕЙ: HUD, меню/выбор/магазин/драфт/пауза/итоги, числа урона
src/pixirender.ts     — pixi/WebGL рендер МИРА и ТЬМЫ (спрайт-пулы + fullscreen-шейдеры)
src/main.ts           — bootstrap: два канваса, фикс-таймстеп цикл, маршрутизация ввода
electron/main.cjs     — Electron main-процесс (грузит dist/ или dev-сервер)
electron/dev.mjs      — dev-цикл: Vite dev-сервер + Electron
tests/                — регресс-харнесс + esbuild-бандл
legacy/               — замороженный ванильный исходник (тег v1.0-vanilla), не редактировать
```

Тюнинг геймплея — только в `src/config.ts`.

## История стека

Игра начиналась как ванильный HTML5 Canvas (один `index.html` + `<script>`-теги, без
сборки). Тех-миграция велась поэтапно, с тегами: `v1.0-vanilla` → `v1.1-typescript`
(TS + Vite ESM) → `v1.2-pixi` (рендер на pixi.js/WebGL) → `v1.3-electron` (десктоп).
Каждый этап сохранял поведение без регрессий (регресс-харнесс зелёный). Оригинал
заморожен в `legacy/`.
